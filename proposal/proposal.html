Project Proposal

Gabe Pinkard, Ian White, Sam Burdick, and Presley Reed III

I. Introduction

We are building an application that allows users to order 3D printed models on demand as well as create/upload and share their own models. Users will be able to log into individual accounts with their own profile information and list of 3D models they have created and/or purchased. Users will also be able to browse the site for models that other users have created/uploaded. Along with the ability to upload models from third parties, users will be able to, in browser, create their own models that are saved and viewable by other users. Likewise users will be able to edit other people’s models and save the new modified models to their profile. Our database will be used not only to store basic user information such as name, userid, age, bio, purchasing history, created models, hashed password, and contact info/social media profiles. The database will also hold all of the model data including price, material, number of purchases, dimensions, category, and creator/editors.

II. Use Cases

  1. Use Case 1: As a user, as long as I am logged in to my account, I can add any model on the site to my cart, and it should stay in the cart until I check out or log in
  2. As a user, so I can purchase the same model again, I can look at my purchasing history and add the item to my cart from their instead of searching for it again.
  3. As a user, as long as I am logged into my account, I want to be able to create a model in my browser and save it. It should be available whenever I visit the site.
  4. As a user who sees a model that I am interested in, I should be able to clone it into my own collection and modify it to my specifications.
  5. As a user, I want to be able to edit my user information such as name and bio information so that I can have a custom unique profile.
  6. As a user, so that I can find models that pertain to my interests, I should be able to view a selection of models pertaining to specific categories.
  7. As a systems admin, to protect our customers information, we should keep all users behind a password wall with an encrypted hash.
  8. As a user, so that I can find a specific model, I should be able to search for it using the model’s name and other relevant attributes.
  9. As a user who wants to find quality products, I should be able to leave and view a rating from 1-5 stars along with a comments about the rating.
  10. As a user, on the homepage, I would like to see a small and interesting assortment of 3D models sorted by category such as featured and popular 3D models.
  11. As a user, I would like to be able to upload a separate .OBJ file and add it to my profile.
  12. As someone who is interested in the 3D models and would like to be able to browse and save my favorites, I should be able to sign up to have a user account with the site and be able to log in and view my saved information.
